Submission layout:
-Project Specification Report: the first submission detailing the goals of our project.
-Project Design Report: most of the database work
-User Manual: A basic outline of how to use the attached files to run the project.
-Other Files: All used to run the project:
	media: Where the actual media is saved
	node_modules: All Node.js downloaded plugins
	views: The EJS files that render the client-side views
	NightOwl.sql: An executable SQL file that creates a copy of our database on your machine
	package.json: A Node.js product that describes project dependencies
	server.js: The Node.js server (run in terminal with "node server.js")
